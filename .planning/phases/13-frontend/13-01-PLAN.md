# Backend API Contracts (from Phase 12)

## GET /api/v1/categories Response (CategoryListResponse)
```typescript
{
  categories: Array<{
    id: string;
    name: string;
    slug: string;
    attribute_schema: {
      year?: boolean;
      make?: boolean;
      model?: boolean;
      // ... other category-specific attributes
    };
    is_active: boolean;
    created_at: string;
    updated_at: string;
  }>;
  total: number;
  page: number;
  page_size: number;
}
```

## POST /api/v1/products Request (CreateProductRequest)
```typescript
{
  title: string;
  price_cents: number;
  category_id: string;
  attributes: {
    vin?: string;  // If present, backend auto-creates vehicle
    year?: number;
    make?: string;
    model?: string;
    trim?: string;
    body_type?: string;
    // ... other vehicle fields
  };
}
```

## POST /api/v1/products Response (ProductResponse)
```typescript
{
  id: string;
  title: string;
  price_cents: number;
  category_id: string;
  attributes: Record<string, any>;
  status: "draft" | "active" | "sold";
  created_at: string;
  updated_at: string;
}
```
