# Plan 12-01 — Category API Completion

## Goal

Wire the `attribute_schema` field (already in DB via c3schema001 migration and on the domain entity)
into the Category DTOs and router. Add missing CRUD endpoints. Enforce role-based active filtering
in the list endpoint.

## Success Criteria

- `POST /api/v1/categories` returns 201 with `attribute_schema` in response body
- `GET /api/v1/categories` returns only `is_active=True` for non-admin users
- `PATCH /api/v1/categories/{id}` updates basic info
- `DELETE /api/v1/categories/{id}` soft-deletes (deactivates)
- `PATCH /api/v1/categories/{id}/attribute-schema` replaces schema with tenant_id check

## Context

**What already exists:**
- `Category` entity: has `attribute_schema: dict[str, Any]` field (line 46), `update_basic_info()`,
  `activate()`, `deactivate()` methods
- `CreateCategoryRequest`: does NOT have `attribute_schema` (gap)
- `CategoryResponse.from_entity()`: does NOT serialize `attribute_schema` (gap)
- Router: has GET list, POST create, GET /{id}, GET /{id}/fields — missing PATCH, DELETE, PATCH /attribute-schema
- `list_categories` handler: passes `is_active` from query param with no role check

**Brain #7 condition:** `UpdateCategoryAttributeSchemaUseCase` MUST verify tenant_id inside the use case, not just the router.

<file>apps/api/src/prosell/application/use_cases/category/update_attribute_schema.py</file>

<description>
New use case (SRP) to REPLACE the attribute_schema on a category.
MUST verify tenant_id inside the use case (Brain #7 Condition 1).
MUST log a warning when replacing a non-empty schema (Brain #7 Condition 2).
</description>
<instructions>
Create UpdateCategoryAttributeSchemaUseCase:
```python
import logging
from typing import Any
from uuid import UUID

logger = logging.getLogger(__name__)

class UpdateCategoryAttributeSchemaUseCase:
    def __init__(self, repo: AbstractCategoryRepository) -> None:
        self.repo = repo

    async def execute(
        self,
        category_id: UUID,
        tenant_id: UUID,
        attribute_schema: dict[str, Any],
    ) -> CategoryResponse:
        category = await self.repo.get_by_id(category_id, tenant_id)
        if not category:
            raise ValueError(f"Category {category_id} not found")
        # tenant_id check: get_by_id already filters by tenant_id — if None returned, access denied
        # Log warning if replacing non-empty schema (data integrity risk)
        if category.attribute_schema:
            logger.warning(
                "Replacing non-empty attribute_schema for category %s (tenant %s). "
                "Existing products with attributes may become invalid.",
                category_id,
                tenant_id,
            )
        category.attribute_schema = attribute_schema
        from datetime import UTC, datetime
        category.updated_at = datetime.now(UTC)
        category = await self.repo.update(category)
        return CategoryResponse.from_entity(category)
```

