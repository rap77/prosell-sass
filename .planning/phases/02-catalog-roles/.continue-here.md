<decisions_made>

**Router Change (Final Solution):**
- Changed user_dealer_router.py from get_current_auth_user_from_cookie to get_current_auth_user
- Reason: get_current_auth_user_from_cookie has internal DB dependencies (jwt_service, user_repository) that FastAPI resolves before overrides
- get_current_auth_user uses Bearer token (Authorization header) which is easier to mock
- All tests updated to use Mock(spec=User) pattern with get_current_auth_user

**Test Strategy:**
- Unit/use-case tests: Use mock repositories (no DB required)
- Integration tests: Mark with skipif, require POSTGRES_AVAILABLE=true
- This allows CI/CD to run without PostgreSQL while enabling full integration tests locally

**Clean Architecture Followed:**
- Domain entities use roles (public), not _roles (private)
- Test fixtures updated to use public interface
</decisions_made>
