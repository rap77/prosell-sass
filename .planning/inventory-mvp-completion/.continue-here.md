<decisions_made>

**Architecture Decisions:**
- **Dealer = Organization** - Multi-niche platform (vehicles, real estate, electronics, etc.)
- **All-or-nothing** - Validación completa antes de cualquier DB write
- **Transaction safety** - async with session.begin() previene datos huérfanos
- **CSV injection protection** - Sanitizar = + - @ con prefix '
- **Rate limiting** - 10 uploads/min, 10MB file size, 1000 rows max
- **Batch VIN lookup** - Single query con IN clause (evita N+1)
- **VehicleWithDealerInfo** - Value object para type safety en dealer mapping
